# Plataforma de Reservas de Canchas Deportivas

## Descripción

Este proyecto consiste en el diseño y desarrollo de una base de datos para la gestión de reservas de canchas deportivas.

Incluye modelado, scripts SQL y documentación funcional del sistema.

---

## Estructura del proyecto

* `docs/` → Documentación, modelo ER, historias de usuario
* `database/` → Scripts SQL (DDL, DML y consultas)

---

## Modelo de Datos

El sistema está basado en un modelo entidad-relación que define entidades como usuarios, reservas, canchas y horarios.

---

## Orden de ejecución

1. Ejecutar el script DDL:

```sql
DDL-Reservas.sql
```

2. Insertar datos:

```sql
DML-Reservas.sql
```

3. Probar consultas:

```sql
Consultas.sql
```
